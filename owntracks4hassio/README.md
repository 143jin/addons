# OwntracksREC

## About ver 0.7.1
Owntracks Recorder Addon for Hassio

## Configuration

config:
```
{
      "localHost": "0.0.0.0",
      "mqttServer": "core-mosquitto",
      "mqttPort": 1883,
      "mqttUser": "username",
      "mqttPassword": "password",
      "topic": "owntracks/#"
  }
```
### Option `mqttUser` (required)

### Option `mqttPassword` (required)
